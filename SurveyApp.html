The Application!

As of now, the application is not on the Apple app store. But, you can access the source of our application at our GitHub repository - HERE.

All you have to do is:

  1. Open up a terminal
  2. cd (directory name) to select the directory that you want to download our source code on
  3. git clone (the ssh key that is given in our repository)
  4. Navigate to the download folder and click Hospital Tabbed App.xcodeproj. This will take you to XCode (make sure to have this install).
  5. Press play and run the iPhone emulation of the app

Make sure the emulation is of iPhone 6. If there are problems using the git clone command, make sure you have install the developer tools provided by XCode. Downloading and opening XCode the first time should prompt one to install the developer tools. Click yes to that prompt and you should be all set.
